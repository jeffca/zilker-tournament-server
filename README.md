# Zilker Apollo Server

The server used by https://zilkerchess.com

## Getting started

Download [NodeJS](https://nodejs.org/en/download/), if you don't have it. (I'm using v16.5.0)

if you don't have yarn, install it
```
npm i yarn -g
```

Install dependencies
```
yarn
```

Start the app
```
npm run start
```
